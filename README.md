# Salary Prediction using Simple Linear Regression

## 📌 Overview
This project applies **Simple Linear Regression** to analyze how an employee’s **Years of Experience** impacts their **Salary**.  
Visualizations include **Scatter Plot** and **Seaborn regplot** to clearly show the trend.
Perform feature encoding.

---

## 📂 Dataset
- **Independent Variable (X):** Years of Experience  
- **Dependent Variable (y):** Salary  

---

## ⚙️ Libraries Used
- Pandas  
- Matplotlib  
- Seaborn (scatterplot, regplot)  
- Scikit-Learn
